Problem Name: Find the Runner-Up Score!

Difficulty: Easy  
Max Score: 10

Problem Statement:
Given the participants' scores, find the runner-up score. The runner-up score is the second highest score in the list.

Input Format:
The first line contains an integer n.  
The second line contains n space-separated integers as a list of scores.

Constraints:
2 <= n <= 10  
-100 <= score <= 100

Output Format:
Print the runner-up score.

Sample Input:
5  
2 3 6 6 5

Sample Output:
5

Explanation:
Given the scores: [2, 3, 6, 6, 5]  
The maximum score is 6, and the runner-up (second highest) score is 5.
